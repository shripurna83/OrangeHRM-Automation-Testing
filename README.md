# OrangeHRM Automation Testing

## Project Overview
This project is an end-to-end automation testing framework developed for the OrangeHRM application.

## Tech Stack
- Java
- Selenium WebDriver
- Cucumber (BDD)
- TestNG
- Maven

## Framework Design
- Page Object Model (POM)
- Cucumber Feature Files
- Step Definitions
- Reusable Utilities
- Explicit Waits

## Modules Automated
- Login Module
- Dashboard Validation
- Recruitment – Add Candidate
- Recruitment – Add Vacancy
- PIM – Employee Management

## How to Execute
1. Clone the repository
2. Import as Maven project in IntelliJ IDEA
3. Run TestRunner.java
